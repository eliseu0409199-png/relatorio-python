# Gerador de Relatório em Python

Projeto desenvolvido em Python para gerar e salvar relatórios de orçamento em arquivo `.txt`.

## 📌 Funcionalidades

- Geração de relatório de orçamento
- Exibição de dados do cliente
- Exibição de produtos e quantidades
- Exibição de valores e mão de obra
- Registro da forma de pagamento
- Criação automática de arquivo `.txt`

## 🛠️ Tecnologias utilizadas

- Python

## ▶️ Como executar

1. Baixe o arquivo
2. Execute o código em um ambiente Python
3. Verifique o arquivo `relatorio.txt` gerado na pasta do projeto
